# Any2Any Project Page - Light Version

This is a cleaner academic project-page variant inspired by BeyondMimic: full-cover teaser media, white content sections, compact abstract, method, results, previews, and BibTeX.

The original dark Any2Any version remains in `../any2any-project-page`.

## Replace the cover with video

Put your teaser at:

```text
assets/video/teaser.mp4
```

Then replace the commented cover snippet in `index.html` with the active `<video>` source block.
